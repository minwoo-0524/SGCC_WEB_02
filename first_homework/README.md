## vscode 사용시 주의사항
1. live server 확장자 설치하기
2. encoding 설정을 utf-8로 해주세요

### encoding 설정 방법
1. 파일 -> 기본 설정 -> 설정
2. 상단의 검색창에 encoding 검색
3. encoding 설정 "utf-8"로 바꿔주기

## live server 실행 방법
1. `ctrl`+`shift`+`P` 를 누른다
2. live server 검색
3. Live Server : Open with Live Server 클릭

## 과제 제출 방법
반드시 **Pull Request** 를 날려주세요
